Brightpeal Extension

Welcome to the Chrome browser Brightpearl Extension! This has been written to slightly modify the functionality of Brightpearl to reduce human error and speed up some processes

To get started, please head over to the Options page to see a description of each modification and decide if to activate them or not.

You can find out more about the extension on GitHub where you are welcome to become part of the project.
